## How to run the project

1. Clone the repository:

```bash
git clone https://github.com/Mihasik556/QR-Code-Project.git
```

2. Open the folder:

```bash
cd QR-Code-Project
```

3. Launch index.html in your browser:

```bash
start index.html
```

Or just open it in the folder
